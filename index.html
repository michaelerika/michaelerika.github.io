一、什么是git

git是一个源代码的管理工具，gitHub是一个网站

二、git常用命令

自报家门：git config --global user.name "你的名字"

自报家门：git config --global user.email "你的邮箱"

初始化一个仓库：git init

  • workspace:工作区
  • stage:暂存区
  • repository:本地仓库
  • remote:远程仓库

添加文件：git add 文件名

版本回退： 命令：git reset --hard 提交的版本号 命令：git reflog 可以查看每一次的提交版本号
